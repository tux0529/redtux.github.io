综合分析能力

新题型

要求对材料中的一句话，或者一个词谈认识，理解，见解，进行解释或阐释

例：21年事业单位统考：谈谈对文中“管理是一门艺术，关键就在于平衡，好比人的体温，36.5℃刚刚好，过高或过低都会影响健康”这句话的认识。（15分）
要求：认识准确，分析全面，层次分明，不超过150字。

答题思路：

  • 是什么（提出观点）
  • 为什么（论证观点）
  • 怎么做（提出对策结论）

是什么：

  1. 如果要求对一句话一个词谈解释，认识，理解 就直接解释这句话的意思；
  2. 如果要求评价某个做法就谈谈这个做法合理或者不合理。

为什么：

根据是什么部分观点的内容进行论证 概括材料中的相应内容作为论据。

例：

观点：双减政策是合理的
论据：概括双减政策的好处，意义等，或不落实双减政策对教育公平产生的危害。
怎么做：就观点或论据中的问题提出直接对策即可，没有直接对策可以不写，不用推导自创对策。

对策类工作

紧扣材料提出对策，参考自创对策五大手段

一、 宣传手段（重点解决人思想意识（心理层面） 以及能力方面的问题 ）

  1. 加大宣传引导（人思想意识）
  • （1）通过电视，网络、新闻媒体 三微一端等形式加大对X的宣传（线上） （覆盖面广）
  • （2）通过讲座，培训，宣讲会等形式，发放宣传手册 入户宣传等形式加大宣传 （提升宣传对象的针对性 ）
  • （3）树立模范典型 通过加大对典型模范先进事迹的宣传 提升XX的意识，发挥模范带动作用
  1. 加大培训 加强培训（能力）
  • （1）发放培训指导手册
  • （2）老带新
  • （3）组织学习交流活动 交流心得
  • （4）开展培训活动
  1. 加大心理疏导 解决人压力大 心理方面的问题，如有怨言，打退堂鼓等。

二、监督手段：（1遏制违法犯罪类问题 2.加强公开方便接受社会监督）

  1. 加大监管力度（违法类问题）。

加大对XX的违法行为的处罚 ，开展针对XX专项行动 畅通监督举报渠道 （信箱 网站等平台）
（注）违法类问题可归纳为监管不到位

  1. 公开透明。

通过官方网站，微信 微博 三微一端等及时发布相关信息接受公众监督 通过举办公众开放日活动让群众参观了解单位工作 听证会 记者招待会，新闻发布会等形式 将XX及时公开。

三、经济手段：解决经济方面的问题

  1. 加大经费投入 完善XX基础设施 改善XX环境 落实XX的优惠政策（吸引人才）
  2. 提高福利待遇 提高XX待遇 出台优惠扶持政策，落实减税降费

四、法律政策手段

  1. 政策的科学性： 政策应能够根据不同地方的情况作出动态调整；
  2. 政策的稳定性： 政策的更新要有过度期，不能朝令夕改；
  3. 普法： 宣传法律；
  4. 法律援助： 法律咨询；
  5. 建议完善： 建立健全XX法律 尽快将X纳入法制轨道。

五、机制体制手段——管理

  1. 完善考核机制。将XX纳入到XX的绩效考核中，奖优罚劣末尾淘汰 调动工作人员的工作积极性；
  2. 完善调休机制。合理制定轮岗轮休制度，明确职责分工合理分配工作。不让工作人员带伤带病上岗；
  3. 制定应急预案。为各类突发情况做好准备（用于解决预防突发情况的发生如：用于景区等公共场所应对客流高峰）；
  4. 出台权利清单。整合部门职能，完善部门协作机制（用于解决部门多头管理职能交叉分散等问题）；
  5. 简化审批程序。简政放权（解决办事审批流程繁琐等问题）；
  6. 出台奖惩措施。出台奖励或惩罚措施以此调动积极性 参与性或引发重视。

人员工作问题类

概括人员工作问题的作答思路

  1. 态度恶劣：对群众冷嘲热讽 刁难群众 语言粗暴
  2. 推卸责任：将自己单位或者自己的责任归咎于他人 其他单位。
  3. 方法不当：(1)与群众沟通方法不当，没有控制好自己的情绪，也没有照顾他人的情绪，没有针对性的进行沟通；(2)工作方法不当。主观臆断，不客观，在没有搞清楚情况的基础上妄下定论，结论，不公正。
  4. 缺乏当担（工作失职）：(1)擅离职守。没有解决好工作问题,，做完工作就离开自己的岗位；(2)违反单位规定。工作流程不遵守单位的规章制度；(3)履职不尽责。没有履行自己的职责按照要求开展工作。

调查调研类工作

答题思路：

调研目标——调研内容——调研对象——调研形式

题目辨别：

（1）写一篇调研报告，调研报告提纲 成绩+问题+对策
（2）写一篇调研要点、调研提纲：1.调研目标+调研对象+调研形式+概括调研内容（有什么写什么）
（3）写一篇调研工作计划、调研工作方案：

  1. 明确调研目标
  2. 明确调研时间，地点，人员
  3. 明确调研对象
  4. 明确调研形式
  5. 明确调研内容

调查调研的基本方法：

明察暗访、发表问卷调查（网络问卷）、访问面谈调查、专家调查、电话调查、新闻媒介调查、文案调查，现场调查等

例题：昆明市交通拥堵现象严重，单位拟对此问题开展进一步调查，请你书写一份调研提纲。300字

  1. 调研目标：本次调研旨在为解决交通拥堵问题提出依据。
  2. 调研对象：交管局相关工作人员、市民
  3. 调研方式：现场实地调查、文案调查、网络问卷调查
  4. 调研内容：1.昆明交通拥堵的地点、时间段 2.导致交通拥堵的原因：拥堵地点是否存在交通违法行为，拥堵地点的道路基础设施建设规范是否完善合理 3.征求群众对拥堵问题的其他意见建议。

培训类工作

作答思路

培训目标——培训对象——培训内容——培训形式，教师

例1：假如你是农机管理中心的工作人员，你要向A乡的农民开展一次农机维修培训，如何开展。

  1. 确定人员：成立本次培训的工作小组，明确分工，下发培训通知，确定培训人员。（有必要的干部入户动员宣传）
  2. 确定培训内容：了解当地农民主要使用的农机类型，收集农民建议和意见汇总确定培训的内容及农机型号。（结合培训对象确定内容）
  3. 确定培训教师：联系邀请农技站相关维修人员，专家作为本次培训的教师，说明培训人员的情况和培训内容。
  4. 明确培训形式：本次培训针对农民的情况采取现场演示的教学形式（现场演示 理论讲解 实操）（因为农民文化程度优先故选择有针对性的培训形式）
  5. 后勤保障：打印准备好教材文件，确定教室场地，调试好教学设备，做好后勤保障。
  6. 明确考勤纪律。完成培训后进行考核，培训中注意收集学员反馈意见。

事件处置

这种问法一般需要事业单位有执法权，开展执法工作：如依法查处假冒伪劣制假贩假，解决处理突发事件如学校食物中毒等。

答题流程：

  1. 成立工作小组制定计划明确分工；
  2. 赶赴现场调查核实情况；
  3. 根据调查情况依法处理事件，要求企业整改，问责责任人；
  4. 对事件处理情况向社会公开；
  5. 形成长效机制避免类似事件再发生；
  6. 总结报告。

纠纷调解类工作

纠纷调解：

要求以工作人员的职能和身份调解矛盾纠纷，受理群众投诉：
作答思路：保持公正客观

答题思路：

公正客观的调解纠纷，以事实为依据

  1. 阅读材料明确纠纷 投诉事件的来龙去脉
  2. 明确纠纷调解涉及到的主体+明确各主体在纠纷中存在的问题，诉求
  3. 针对不同纠纷主题就其问题提出对策，提出对策满足其诉求
  4. 如果该纠纷可以在今后避免 提出整改的措施避免类似情况的再次出现

答案写法：

  1. 成立工作组到现场调查;
  2. 与不同纠纷主体进行沟通解决问题提出对策满足诉求；
  3. 提出对策避免类似事件的发生；
  4. 总结报告，反馈处理情况。
