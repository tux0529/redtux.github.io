综合分析能力

新题型

要求对材料中的一句话，或者一个词谈认识，理解，见解，进行解释或阐释

例：21年事业单位统考：谈谈对文中“管理是一门艺术，关键就在于平衡，好比人的体温，36.5℃刚刚好，过高或过低都会影响健康”这句话的认识。（15分）
要求：认识准确，分析全面，层次分明，不超过150字。

答题思路：

  • 是什么（提出观点）
  • 为什么（论证观点）
  • 怎么做（提出对策结论）

是什么：

  1. 如果要求对一句话一个词谈解释，认识，理解 就直接解释这句话的意思；
  2. 如果要求评价某个做法就谈谈这个做法合理或者不合理。

为什么：

根据是什么部分观点的内容进行论证 概括材料中的相应内容作为论据。

例：

观点：双减政策是合理的
论据：概括双减政策的好处，意义等，或不落实双减政策对教育公平产生的危害。
怎么做：就观点或论据中的问题提出直接对策即可，没有直接对策可以不写，不用推导自创对策。

对策类工作

找出材料直接对策

  1. 标志性词汇：XX建议、提议、要、应、需、需要、应该、必须。。动词开头
  2. 他人的成功经验（获得了成功的做法）
  3. 成效

找出材料间接对策

由问题推对策、由原因推对策 （反着来）–仅适用于非违法犯罪的问题

违法类问题可归纳为监管不到位

紧扣材料自创对策（七大手段）

一、宣传手段：（针对思想层面）：

  1. 加大宣传引导。①通过电视，网络、新闻媒体、三微一端等形式加大对X的宣传 （覆盖面广）；②通过讲座，培训，宣讲会等形式，发放宣传手册 入户宣传等形式加大宣传 （提升宣传对象的针对性 ）；③树立模范典型。通过加大对典型模范先进事迹的宣传，提升XX的意识，发挥模范带动作用。
  2. 加大心理疏导。邀请心理咨询师或以倾听，交流，鼓励等形式排解工作人员的心理压力

二、制度手段：（针对人员管理，对单位内部）

  1. 完善考核机制。将XX纳入到XX的绩效考核中，奖优罚劣末尾淘汰，调动工作人员的工作积极性。
  2. 完善调休机制。合理制定轮岗轮休制度，明确人员职责分工，不让工作人员带伤带病上岗。
  3. 制定应急预案。为XX突发情况做好准备。（用于解决预防突发情况的发生如：用于景区等公共场所应对客流高峰）
  4. 加强统筹协调。出台权利清单，整合部门职能，各部门成立领导小组，（用于解决部门多头管理职能交叉分散等问题）
  5. 建立首问责任制。要求窗口首问人员全程协调处理，跟踪群众问题，一次性办结，避免推诿扯皮。
  6. 简化办事流程。简化审批程序，简政放权。（解决办事审批流程繁琐等问题）
  7. 出台奖惩机制。①出台制度并明确责任人，问责追究制度落实不到位行为，对认真执行制度、按规矩办事的人给与表彰奖励。②建立奖励机制，激励单位员工或社会群众参与到某项工作中来。

三、监督（监管）手段：（加强单位内部及外部的监督管理）

  1. 加大监管力度（针对违法类问题）。加大对XX的违法行为的处罚，开展针对XX专项行动，多部门联合查处，整治XX行为，畅通监督举报渠道。（信箱 网站等平台）
  2. 完善反馈机制。建立线上（市长热线、政府邮箱），线下（信访投诉）反馈渠道，收集民意分析研判，及时上报解决问题。
  3. 公开透明。通过官方网站，微信、微博、三微一端等及时发布相关信息接受公众监督； 通过举办公众开放日活动让群众参观了解单位工作；通过听证会、记者招待会、新闻发布会等形式将XX及时公开。
  4. 建立好差评制度。在各级政务服务机构开展好差评，社会群众广泛评价，督促政府整改，形成良性互动。

四、服务手段：（针对服务对象，对单位外部）

  1. 完善法律服务。对于社会弱势群体定期组织开展法律援助、法律咨询等服务。
  2. 精细服务。开展网格化管理，分网格指派网格员及时解决问题，了解民情。针对特殊群体，困难企业开设绿色通道。
  3. 开设线上服务。借助微信群，收集群众反馈的问题，了解基层情况。提供一对一线上指导业务办理服务。

五、人才手段

  1. 组织开展培训（提升人员工作能力水平）①明确工作流程，发放工作指导手册；②通过老带新的形式传授工作经验，让新员工尽快适应工作；③定期组织学习交流活动，交流心得体会；④组织技能大比武，先进评选等培训活动。⑤开展网络视频培训。
  2. 加强组织建设。从XX专业组织或老党员，老干部中吸纳能人骨干成立XX组织，并发挥其引领示范作用。
  3. 发动社会力量。组织发动社会单位或群众参与到XXX管理中，实现协同共治。

六、资金手段（解决经济方面的问题）

  1. 加大经费投入。完善XX基础设施 改善XX环境 落实XX的优惠政策（吸引人才）提高福利待遇提高XX待遇。
  2. 出台优惠扶持政策。落实减税降费，加大补贴力度，开辟绿色通道。
  3. 完善经费支撑。设立专项资金，通过上级划拨、金融支持、吸引社会资本等形式解决资金来源不足的问题。

七、技术手段

  1. 科学制定工作方案。在协调资金、人力及项目前开展调研工作，听取专家评估意见，科学规划施策，分配资金。
  2. 技术赋能。开通线上申请办理、打造智能平台、引入智能技术，智能机械等手段方便群众或企业办事，提升办事效率，省时省力，实现智能化管理。

人员工作问题类

概括人员工作问题的作答思路

  1. 态度恶劣：对群众冷嘲热讽 刁难群众 语言粗暴
  2. 推卸责任：将自己单位或者自己的责任归咎于他人 其他单位。
  3. 方法不当：(1)与群众沟通方法不当，没有控制好自己的情绪，也没有照顾他人的情绪，没有针对性的进行沟通；(2)工作方法不当。主观臆断，不客观，在没有搞清楚情况的基础上妄下定论，结论，不公正。
  4. 缺乏当担（工作失职）：(1)擅离职守。没有解决好工作问题,，做完工作就离开自己的岗位；(2)违反单位规定。工作流程不遵守单位的规章制度；(3)履职不尽责。没有履行自己的职责按照要求开展工作。

调查调研类工作

答题思路：

调研目标——调研内容——调研对象——调研形式

题目辨别：

（1）写一篇调研报告，调研报告提纲 成绩+问题+对策
（2）写一篇调研要点、调研提纲：1.调研目标+调研对象+调研形式+概括调研内容（有什么写什么）
（3）写一篇调研工作计划、调研工作方案：

  1. 明确调研目标
  2. 明确调研时间，地点，人员
  3. 明确调研对象
  4. 明确调研形式
  5. 明确调研内容

调查调研的基本方法：

明察暗访、发表问卷调查（网络问卷）、访问面谈调查、专家调查、电话调查、新闻媒介调查、文案调查，现场调查等

例题：昆明市交通拥堵现象严重，单位拟对此问题开展进一步调查，请你书写一份调研提纲。300字

  1. 调研目标：本次调研旨在为解决交通拥堵问题提出依据。
  2. 调研对象：交管局相关工作人员、市民
  3. 调研方式：现场实地调查、文案调查、网络问卷调查
  4. 调研内容：1.昆明交通拥堵的地点、时间段 2.导致交通拥堵的原因：拥堵地点是否存在交通违法行为，拥堵地点的道路基础设施建设规范是否完善合理 3.征求群众对拥堵问题的其他意见建议。

培训类工作

作答思路

培训目标——培训对象——培训内容——培训形式，教师

例1：假如你是农机管理中心的工作人员，你要向A乡的农民开展一次农机维修培训，如何开展。

  1. 确定人员：成立本次培训的工作小组，明确分工，下发培训通知，确定培训人员。（有必要的干部入户动员宣传）
  2. 确定培训内容：了解当地农民主要使用的农机类型，收集农民建议和意见汇总确定培训的内容及农机型号。（结合培训对象确定内容）
  3. 确定培训教师：联系邀请农技站相关维修人员，专家作为本次培训的教师，说明培训人员的情况和培训内容。
  4. 明确培训形式：本次培训针对农民的情况采取现场演示的教学形式（现场演示 理论讲解 实操）（因为农民文化程度优先故选择有针对性的培训形式）
  5. 后勤保障：打印准备好教材文件，确定教室场地，调试好教学设备，做好后勤保障。
  6. 明确考勤纪律。完成培训后进行考核，培训中注意收集学员反馈意见。
